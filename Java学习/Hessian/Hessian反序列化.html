Hessian反序列化

Hessian1.0

HessianInput#readObject方法,由于Hessian会将序列化的结果处理成一个Map(自定义对象)，所以序列化结果的第一个byte总为M（ASCII为77）

接着会进入ObjectInputStream#readMap通过getDeserializer()来获取一个deserializer
在获取到deserializer后，java会创建一个HashMap作为缓存，并将我们需要反序列化的类作为key放入HashMap中。

自定义对象默认采用UnsafeDeserializer#readMap

方法使用Unsafe.allocateInstance方法直接还原对象
紧跟是还原对象属性信息，同样会有相应的fieldDeSerializers还原变量

反序列化

对于map类型的对象使用MapDeserializer#readMap

首先会创建一个Map对象，然后将key和value分别反序列化put进map中

调用hash时会调用key.hashcode

选择一条以hashcode入口的反序列化链即可
利用signObject打二次反序列化
ToStringBean.toString()可以调用getter方法

    public static byte[] Serialize(Object o) throws IOException {  
        ByteArrayOutputStream byteArrayOutputStream = new ByteArrayOutputStream();  
  
        HessianOutput hessianOutput = new HessianOutput(byteArrayOutputStream);  
        hessianOutput.writeObject(o);  
        hessianOutput.flush();  
  
        return byteArrayOutputStream.toByteArray();  
    }  
  
    public static Object Deserialize(byte[] bytes) throws IOException {  
        ByteArrayInputStream byteArrayInputStream = new ByteArrayInputStream(bytes);  
  
        HessianInput hessianInput = new HessianInput(byteArrayInputStream);  
        return hessianInput.readObject();  
    }
